Stock Analyzer


Overview

This is a tool that I have created to gather critical information about any given stock
(so long as it is on Yahoo Fiance) and returns it to the user in a readable and not so
technical way, because people like to complicate financial terms. When in reality, these
things are very simple to understand.

I built this tool using a handy webscraping tool called Selenium, which uses the HTML code within a given website to gather information about whatever you want or need. For my particular version I used python to build my analyzer and the corresponding Python version of Selenium.



How to Use It

Below this block will be a box where you can type the ticker of the stock you would like the information on. Once you have typed in the stock ticker, click the submit button and the stock info and explanation for how it was retrived and processed will be returned.
