Stock Analyzer


Overview

This is a tool that I have created to gather critical information about any given stock
(so long as it is on Yahoo Fiance) and returns it to the user in a readable and not so
things are very simple to understand.

I built this tool using a handy webscraping tool called Selenium, which uses the HTML code within a given website to gather information about whatever you want or need. For my particular version I used python to build my analyzer and the corresponding Python version of Selenium.



How to Use It

Below this block will be a box where you can type the ticker of the stock you would like the information on. Once you have typed in the stock ticker, click the submit button and the stock info and explanation for how it was retrived and processed will be returned.

Price to Earnings Ratio {{PE_ratio}} This ratio is used specifically to relate the price of a stock at a given time to its earnings per share. Beta Value {{beta}} The beta value of a stock is a value calculated to model the risk or volatility of a stock based on the fluctuation of its price over a certain period of time.

Percentage Held by Institutions {{institution_hold}} This shows what percentage of outstanding shares are held by financial institutions.
Number of Shares Short {{shares_short}}% Shares short shows what percentage of outstanding shares are held in a short position. Which means the people holding these shares are betting the stock is going to go down in price over a certain amount of time.
Dividend {{dividend}} This is an option a publically traded company has to give a set percentage payout to shareholders as an incentive to keep the stock and buy more. Stocks that offer dividends are typically low risk.
